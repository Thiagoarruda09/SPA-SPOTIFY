# SPA-SPOTIFY
Single Page Application baseado no spotify.

## Tecnologias utilizadas
- HTML
- CSS
- javacript
- node.js
  ## Ferramentas utilizadas
  - NPM
  - JSON Server
### instalação
- dar o seguinte comando no terminal : json-server --watch api-artists/artists.json port 3000
